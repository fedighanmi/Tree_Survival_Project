Introduction#

tree_lab is a Python package designed to simplify dataset analysis for users with minimal technical expertise in a specific use case with a specific dataset. The goal of this project is to provide a user-friendly functions for basic dataset analysis. It includes functionalities for performing basic preprocessing, cleaning and visualizing results.
